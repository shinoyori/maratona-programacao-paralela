# maratona-programacao-paralela
Maratona de Programação: Implementação em C/OpenMP de algoritmos paralelos para otimização de alto desempenho. O projeto aborda 4 problemas clássicos: Otimização por Colônia de Formigas (TSP), N-Queens, Simulação de Onda Acústica 3D (FDM) e Resolução de Sistemas Tridiagonais (RC). Foco em técnicas de paralelismo e speedup.
